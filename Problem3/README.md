# Asterisk Game – HTML & JavaScript Version

## Overview

Asterisk is a fast-paced, single-player arcade-style game where the player must carefully navigate through obstacles by controlling vertical movement. This project is a web-based port of an original C# WinForms implementation, recreated using HTML and JavaScript.

The goal of the game is to survive as long as possible by passing through gaps in incoming obstacles while progressing through increasingly difficult levels.

---

## How to Play

* Press **Enter** to move upward.
* Release **Enter** to fall downward.
* Navigate through the gap on the right side of the screen to advance to the next level.
* Avoid hitting walls or obstacles.

### Game Over

* Colliding with any obstacle or wall will end the game.
* A **Game Over** message will be displayed.
* The player can restart the game and try again.

---

## Features

### Core Gameplay

* Smooth vertical movement controlled by a single key
* Progressive level system
* Collision detection with obstacles

### Visual Enhancements

* Dynamic color changes for improved visual feedback
* Real-time score counter and level display
* Gap moves and shrinks as levels increase
